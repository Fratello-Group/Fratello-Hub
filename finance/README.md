# Finance

Financial dashboards, costing tools, reports, and margin analysis for the Fratello controller and ownership team.

## Subfolders

- **dashboards/** — Financial reporting dashboards (P&L, cash flow, KPIs)
- **tools/** — Costing calculators, margin analyzers, pricing tools
- **reports/** — Report templates and generators

## Access

Most finance tools are Tier 3 (Owner). Department-level tools may be Tier 2 (Manager).

## Owner

Controller
