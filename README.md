# Fratello Ops Hub

**Fratello Group Inc.** — Centralized operational intelligence platform.

This repository contains all tools, dashboards, forms, reports, and workflows used by the Fratello team. The Hub is a branded web portal hosted on GitHub Pages where team members access role-appropriate tools organized by department.

---

## Quick Start

1. Clone this repo
2. Open `system/hub/index.html` in a browser to preview the portal
3. See [CONTRIBUTING.md](docs/CONTRIBUTING.md) for how to add new tools
4. See [TOOL-REGISTRY.md](docs/TOOL-REGISTRY.md) for the master list of all tools

---

## Structure

```
fratello-hub/
├── finance/          Financial dashboards, costing tools, reports
├── production/       Roasting, packaging, warehouse, planning, safety
├── sales/            Sales intelligence, CRM, forecasting
├── marketing/        Brand assets, catalogs, promotional tools
├── hr/               Performance, wages, onboarding, policies
├── operations/       Forms, SOPs, legal, company-wide processes
├── system/           Hub portal, shared components, API integrations
└── docs/             Vision, architecture, contributing guide, registry
```
