Claude Design Skill

Make Fratello work look like Fratello.

This is Mateo's Fratello design skill for Claude. Upload it into a Claude project so Claude knows the Fratello brand system before creating documents, decks, PDFs, social content, or other visual work.

Download Skill File Open Claude

First-time setup

  1. Go to Claude and open the Fratello project you want to use.
  2. Choose Add Content, then upload the downloaded skill file.
  3. If an older version is already there, delete the old one first.

The skill file is the brand brain. Once uploaded, Claude can use it in that project.

How to use it

  • Ask in plain language, like "make a one-pager for Godfather Espresso."
  • Upload any photos you want Claude to use at the start of the chat.
  • Do not attach logos unless needed; the skill already includes logo guidance.

Claude cannot pull from Google Drive by itself, so upload the actual images you want used.
