# System

Infrastructure that makes the Hub work. Maintained by whoever is architecting the system, not by department users.

## Subfolders

- **hub/** — Portal entry point, navigation, authentication
- **components/** — Reusable UI components shared across tools
- **api/** — Integration middleware (Odoo, Shopify, Google Sheets connectors)
- **design-tokens/** — Fratello brand colors, fonts, spacing variables
- **skills/** — AI skill files used across departments
