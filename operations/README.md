# Operations

Company-wide forms, standard operating procedures, legal documents, and cross-departmental processes.

## Subfolders

- **forms/** — Company-wide forms (fillable, trackable)
- **manuals/** — SOPs, procedures, operational docs
- **legal/** — Contracts, compliance, insurance, supplier agreements

## Access

Forms and manuals are Tier 1 (Staff). Legal is Tier 3 (Owner).
